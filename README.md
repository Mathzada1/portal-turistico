# portal-turistico
Nome do projeto: Brasil Encantado
Participantes: Matheus Segatto Gardim

Portal turístico "Brasil Encantado" desenvolvido para o projeto acadêmico 2026. O site explora destinos nacionais e gastronomia típica através de uma interface moderna e responsiva. Tecnologias utilizadas: HTML5 e CSS3 (Vanilla), aplicando conceitos de Flexbox, efeitos de Parallax fixo, submenus interativos e design adaptável.
